## Module Organization

### HOSDR210 — Driver Module

**Purpose:** Orchestrates claim processing — provider lookup, wage index lookup, and pricer invocation
**Location:** `/HOSDR210`
**Key sections:**
- LINKAGE SECTION: Defines BILL-315-DATA contract (315 bytes I/O)
- PROV-TABLE: In-memory provider table (2,400 entries × 3 segments of 80 bytes)
- MSA-WI-TABLE: Legacy MSA wage index table (4,000 entries)
- CBSA-WI-TABLE: CBSA wage index table (9,000 entries)
- PROCEDURE DIVISION: FY date boundary calc → provider search → wage index search → CALL HOSPR210

### HOSPR210 — Pricer Module

**Purpose:** Core payment calculation engine — all business rules for FY1998 through FY2021
**Location:** `/HOSPR210`
**Key sections:**
- WORKING-STORAGE: Work variables, payment rates, RHC logic flags, date calculation fields
- `COPY HOSPRATE`: Includes external rate copybook
- LINKAGE SECTION: Same BILL-315-DATA as driver
- PROCEDURE DIVISION: FY routing → per-FY processing paragraphs → revenue code calculations

### HOSPRATE — Rate Copybook

**Purpose:** Externalized payment rate constants for FY2016 through FY2021
**Location:** `/HOSPRATE`
**Key data:** Per-FY rate records, each containing 8 rates (4 care types × LS/NLS) plus QIP-adjusted variants. Rates prior to FY2016 are hard-coded directly in HOSPR210.

### CBSA2021 — Wage Index Reference Data

**Purpose:** Core-Based Statistical Area wage index values for geographic payment adjustment
**Location:** `/CBSA2021`
**Format:** Fixed-width positional — `CBSA(5) + EffDate(8) + WageIndex(6) + Filler(6) + AreaName(variable)`
**Coverage:** ~500 CBSAs × ~15 fiscal years (FY2006–FY2021)

### TESTJCL — Test Job Control

**Purpose:** Batch execution job for testing the pricer end-to-end
**Location:** `/TESTJCL`
**Key steps:**
- STEP02: Delete prior output datasets
- RUN: Execute HOSOP210 with all input files (BILLFILE, PROVFILE, CBSAFILE, MSAFILE)
- LIST: SAS step to read RATEFILE output and produce formatted report

### test/ — Local Test Harness

**Purpose:** GnuCOBOL-based local test environment replacing the mainframe CICS/JCL pipeline
**Location:** `/test/`
**Contents:**
- `HOSOP210.cbl` — Local test driver (replaces production CICS wrapper): opens files, loads tables, invokes HOSDR210
- `GENDATA.cbl` — Test data generator: creates PROVFILE (2 providers) and BILLFILE (40 test cases)
- `build-and-run.bat` — Automated 7-step build and execution script
- `README.md` — Setup and usage instructions

### run/ — Runtime Data

**Purpose:** Generated and reference data files used by the test harness
**Location:** `/run/`
**Contents:**
- `PROVFILE` — Generated by GENDATA (2 providers: 341234 Charlotte NC, 341235 NYC)
- `BILLFILE` — Generated by GENDATA (40 test bills: TC01–TC36 + TC08B)
- `CBSAFILE` — Copy of CBSA2021 (7,478 CBSA wage index records)
- `MSAFILE` — Manual creation (1 record: MSA 6740, WI=1.0000)
- `RATEFILE` — Output: priced claim records

## Where Things Live

**Payment calculation logic:** HOSPR210 (PROCEDURE DIVISION, FY-specific paragraphs)
**Rate constants (FY2016+):** HOSPRATE (COPY member)
**Rate constants (pre-FY2016):** HOSPR210 (hard-coded in each FY paragraph)
**Data lookups (provider, wage index):** HOSDR210 (table SEARCH logic)
**Reference data:** CBSA2021 (wage index flat file)
**Job execution & reporting:** TESTJCL (mainframe), test/build-and-run.bat (local)
**Local test programs:** test/HOSOP210.cbl (driver), test/GENDATA.cbl (data generator)
**Runtime data:** run/ (PROVFILE, BILLFILE, CBSAFILE, MSAFILE, RATEFILE)
**Missing from repo:** Production HOSOP210 (CICS wrapper — different from test/HOSOP210.cbl)
